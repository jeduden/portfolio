A-Star

This code demonstrates the A-star path finding algorithm.

It displays a map with random obstacles. A click on an empty tile on the map sets the start point. A second click on an empty tile determines the end point of the path.

The A-Star implementation is written in client-side JavaScript with a heap as priority queue. The implementation is not optimized for speed.
